# Python-Rock-Paper-Scissors
A game written in Python that lets you play Rock Paper Scissors against your computer.
## Features
- Text formatting
- Option to play again
- Score tracking
## Requirements
### Windows version (.exe)
- A computer running Windows, preferably Windows 10 or later
### Raw Python file (.py)
- Python 3.6+ (3.13 reccomended)
- 'colorama' module (install via pip)

## Installation
### Windows version
1. Download RockPaperScissors.exe from releases
2. Open RockPaperScissors.exe
### Raw Python file
1. Install Python at python.org
2. In your terminal, enter
   > pip install colorama
4. Download RockPaperScissors.py
5. Open RockPaperScissors.py
